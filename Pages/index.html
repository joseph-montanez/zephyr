01 · BUILT FOR THE METAL

Stop waiting on QCAD. Stop paying AutoCAD.

Most cross‑platform CAD is Qt running on top of OpenGL pretending to be modern. Zephyr uses Direct3D 12 on Windows and Metal on macOS through a single Swift GPU pipeline — so it actually feels native on both.

macOS
Metal · MetalKit · SwiftUI menu bar
  • ✓ Universal binary, Apple silicon native
  • ✓ Real Cmd shortcuts, real trackpad gestures
  • ✓ High DPI on every external display
  • ✓ Files app, Quick Look, drag‑and‑drop
Windows
Direct3D 12 · SDL3 GPU · x64 + ARM64
  • ✓ Native ARM64 — runs natively on Snapdragon & Surface
  • ✓ No .NET install, no Visual C++ runtime
  • ✓ Single signed .exe, ~14 MB
  • ✓ Per‑monitor DPI, multi‑display ready
02 · CLEAN SPECKLES

The dirty‑DXF janitor you've always wanted.

Scanned a paper drawing? Imported a forty‑year‑old DXF? You know the speckles — tiny stray dots and dashes scattered through the geometry.

Point Zephyr at one. It learns the color and size signature, finds every match across the drawing, and lets you erase them in one motion. No regex. No script. No half‑day cleanup.

03 · COMPATIBILITY

Opens the files everyone else hands you.

DWG and DXF — round‑trip ready, including blocks, layers, hatches, splines/NURBS, and polylines with bulge. PDF export ships a Bluebeam‑compatible Measurement dictionary so dimensions survive the handoff.

DWG
IMPORT
Full read via libdxfrw bridge. All entity types, blocks, layers, text styles, hatches.
DXF
IMPORT EXPORT
R2007 round‑trip. Dimensions, splines, hatches, leaders — all preserved.
PDF
EXPORT
Vector PDF 1.7 with Bluebeam Revu‑compatible Measurement dictionary.
EAB
IMPORT EXPORT
Fast native binary. BVH spatial index, zstd compression, viewport‑culled partial loads.
SHX vector text included. Full shape‑font byte‑code interpreter — AutoCAD .shx files render as pure vectors at any zoom. Bitmap text degradation is a problem from 2009.
